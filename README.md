🎬 Movie Recommendation System

A content-based movie recommendation system built using Python, Pandas, Scikit-learn, and Streamlit that suggests similar movies based on user input.

🚀 Features
Movie recommendations
Content-based filtering
Cosine similarity
Streamlit web app

🛠️ Tech Stack
Python
Pandas
NumPy
Scikit-learn
Streamlit

▶️ Run the Project
Clone Repository
git clone https://github.com/your-username/movie-recommendation-system.git
Install Requirements
pip install -r requirements.txt
Run App
streamlit run app.py
